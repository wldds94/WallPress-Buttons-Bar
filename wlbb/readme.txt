The best Button Bar Management System. The solution for link in custom CPT like product of "WooCommerce" & more.

== Description ==
Easily create and manage a button links on your WordPress site with [WordPress Button Bar] free plugin.
Wlbb provide you a button manager to create your button e filter your Bar for each page, post, CPT, product ecc.

Full list of sections and features of plugin

* Custom Buttons
* Dynamic Custom Buttons Link
* Filter by single page CPT & more
* Dynamic & Customizable links for buttons

= Privacy notices =

With the default configuration, this plugin, in itself, does not:

* write any user personal data to the database;
* send any data to external servers;
* use cookies.

== Installation ==

1. Upload the entire `wlbb` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).

You will find **Button Manager** menu in your WordPress admin screen.
